## 2025-05-18 - SPF Tree Interactive Keyboard Navigation\n**Learning:** The SPF tree view utilizes span elements disguised as expandable list headers. Mouse events were previously handling toggles, but these items lacked native keyboard support or accessible properties, creating a trap for keyboard-only users navigating the report.\n**Action:** Apply `role="button"`, `tabindex="0"`, and manage the `aria-expanded` state dynamically in JavaScript. Bind both `Enter` and `Space` keys alongside click handlers for custom accessible controls.
